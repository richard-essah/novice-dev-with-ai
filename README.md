# Web Development + AI Engineering Classroom

Learn modern web development with HTML, CSS, JavaScript, PHP, MySQL, and AI-assisted workflows using Claude, GitHub Copilot, and Antigravity IDE.

This repository is the official classroom workspace for the course (HTML+CSS).

---

# 📚 What You Will Learn

By the end of this course, students will be able to:

- **Build websites with HTML & CSS**
- **Create responsive layouts with Flexbox & Grid**
- **Add interactivity using JavaScript**
- **Use AI tools professionally in software development**
-** Understand modern AI-assisted coding workflows**

---

# 🧠 AI-First Development Workflow

In this class, we are not just learning coding.

We are learning how modern developers build software using:

- Claude
- ChatGPT
- GitHub Copilot
- Antigravity IDE
- Context engineering
- AI skills systems
- Prompt-driven development

Students will learn how to:
- Use AI responsibly
- Debug code with AI
- Generate boilerplate
- Understand code deeply
- Structure projects professionally
- Build faster without depending blindly on AI
